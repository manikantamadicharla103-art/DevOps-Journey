# Day 01 - Linux Basics (Fedora)

## Objective

Today I started my DevOps journey by learning the basic Linux commands on Fedora. I understood how to navigate the Linux file system and create files and directories using the terminal.

---

## Topics Covered

### 1. Linux

Linux is a free and open-source operating system widely used for servers, cloud computing, DevOps, and software development.

### 2. Terminal

The terminal is a command-line interface (CLI) that allows users to interact with the operating system.

### 3. Linux File System

Common directories:

- `/`
- `/home`
- `/etc`
- `/var`
- `/usr`
- `/tmp`
- `/root`

---

## Commands Learned

### 1. pwd

Displays the current working directory.

```bash
pwd
```

Example output:

```text
/home/manikanta
```

---

### 2. ls

Lists files and directories.

```bash
ls
ls -l
ls -a
```

---

### 3. mkdir

Creates a directory.

```bash
mkdir DevOps
```

---

### 4. cd

Changes the current directory.

```bash
cd DevOps
cd ..
cd ~
```

---

### 5. touch

Creates a new file.

```bash
touch linux.txt
```

---

## Practical Exercise

```bash
pwd
ls
mkdir DevOps
cd DevOps
mkdir Linux
cd Linux
touch linux.txt docker.txt aws.txt
ls
pwd
```
